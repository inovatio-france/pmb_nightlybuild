Icònas utilizadas dins l'OPAC

Icònes qu'indican los periodics :
  •  article
  •  periodic
Icònas de la mena de supòrt :
  •  tèxt imprimit
  •  tèxt manuscrich
  •  particion musicala imprimida
  •  particion musicala manuscricha
  •  document cartografic imprimit
  •  document cartografic manuscrich
  •  document projectat o vidèo
  •  enregistrament sonòr non musical
  •  enregistrament sonòr musical
  •  document grafic de doas dimensions
  •  document electronic
  •  document multimèdia
  •  objècte de 3 dimensions, artefactes, ...

Las monografias son representadas sonque per l'icòna de lor mena de supòrt.
Exemple :  Monografia subre tèxt imprimit (libre)
