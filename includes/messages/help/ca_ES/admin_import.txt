<h2>Administraci&oacute; de les importacions</h2>
<p>Aquest apartat permet generar les importacions de dades associats a les obres
o als exemplars.</p>
<p>El procediment es realitza en diferents passos : definici&oacute; del prestador 
(ex : biblioteca departamental de pr&eacute;stec), importaci&oacute; dels registres o 
importaci&oacute; dels exemplars (i dels registres associats), i finalment marcar 
les importacions per verificar la correspond&egrave;ncia entre el llistat de pr&eacute;stec i 
les obres realment transmeses.</p>
<p>Aquesta darrera verificaci&oacute; &eacute;s r&agrave;pida i pot &eacute;sser integrament realitzada amb 
un lector &ograve;ptic.</p>
